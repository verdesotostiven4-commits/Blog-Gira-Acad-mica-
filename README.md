# Blog Académico de la Gira de Biodiversidad

Sitio web estático para presentar la gira académica 2026 sobre biodiversidad de bosques montanos.

## Contenido incluido

- Inicio general de la gira académica.
- Apartado de Flora con órdenes, familias y fichas resumidas.
- Apartado de Fauna con órdenes, familias y fichas resumidas.
- Galería dividida en flora y fauna, con filtros por hábito o grupo.
- Guía interna para replicar el diseño en Google Sites.

## Base documental

El contenido se armó a partir de los informes entregados de Flora y Fauna. Los datos principales están en `app.js` dentro de `DATA_TEXT`.

## Despliegue en Vercel

1. Entrar a Vercel.
2. Importar este repositorio desde GitHub.
3. Framework: Other / Static.
4. Build command: dejar vacío.
5. Output directory: dejar vacío o `/`.
6. Deploy.

## Cómo usarlo para Google Sites

Usa este sitio como maqueta visual. En Google Sites crea páginas: Inicio, Flora, Fauna y Galería. Dentro de Flora y Fauna crea subpáginas por orden y luego por familia. En cada familia inserta las fichas correspondientes y en Galería sube las fotos con pie de imagen: nombre común + nombre científico.
